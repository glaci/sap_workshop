Business Scenario

You’re a IT manager for a new business unit (BU-S) in ABC company. Recently, you were assigned a task to set up a landing zone for end-users. The target is to setup an environment for users to access SAP systems in BU-M. Besides, due to Covid-19, some users need to use WorkSpaces. Since you’re new to some AWS services, this workshop will guide you to go through this task.

Technical Requirement - Summary

You have to prepare the environment to

  • Connect SAP S/4 HANA system in BU-M (seperate AWS account)
  • Manage standalone Active Directory (AD) in BU-S to join BU-M and create users in BU-S
  • Deploy WorkSpaces to users with single sign-on (SSO) feature
