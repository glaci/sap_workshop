SAP GUI SSO Congiguration

  • Log into SAP system – double click SAP Logon in the desktop

  • Right click and Add New Entry -> Connection -> User Specified System

  • Description: SAP System

  • System ID: NWD

  • Instance Number: 66

  • SAProuter String: (Empty)

  • Application Server: 10.0.1.50

  • Click Finish

  • Double click to log in, User is Admin temporally

  • Type T-code SU01 to create you own user

  • Copy user from Admin to you

  • Enter your password and save

  • Exit SAP GUI and log into the system with new user

  • Get back to SU01, and configure SNC to your domain user and save

  • Now let’s test SSO feature – right click the item -> Properties

  • Double click the SAP system again – SSO works
