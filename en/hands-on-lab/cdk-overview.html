CDK Overview

This CDK sample code deploys 4 Stacks for the environment including,

  • A Prep Stack for Transit Gateway and domain info
  • A VPC Stack to build up network environment
  • A Directory Service Stack to build up AD Connector
  • A WorkSpaces Stack to build up WorkSpaces
