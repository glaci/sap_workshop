Preparation

Prerequisite

  • Create AWS Account in Tokyo (ap-northeast-1)
  • AWS Command Line Interface (AWS CLI) should be already configured with Administrator permission in .aws/credentials
  • Python 3 should be installed
  • Higher than version 1.33.0 of Amazon Cloud Development Kit (AWS CDK) should be installed.
  • git tool already installed
  • Access permission to Amazon Code(VPN required)
  • Provide your AWS Account to Hank

Environment Set up

  • Clone git repository to your local environment
$ git clone ssh://git.amazon.com/pkg/AWSWorkSpacesCrossVPC
  • Navigate to AWSWorkSpacesCrossVPC folder
$ cd AWSWorkSpacesCrossVPC
  • Create key pair for EC2 instance and remote desktop
$ python3 createKeyPair.py <AWS Profile>

This script is to

  • Create a key pair with name keyWorkspace
  • Download the key pair file into your folder (key_workspace.pem)
